## Expense Intelligence

Expense Intelligence is a web-based dashboard for tracking personal or business spending. Log new expenses, slice by category or timeframe, and visualize where your money is going.

### Features
- Snapshots of total, recurring, and average spend that update with filters
- Dynamic filtering by text, category, frequency, and date range
- Inline category and monthly breakdowns to reveal spending patterns
- Quick-add modal for capturing new expenses with optional notes
- Local storage persistence so data survives refreshes

### Getting Started
```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) and start exploring the dashboard.

### Production Build
```bash
npm run build
npm start
```
